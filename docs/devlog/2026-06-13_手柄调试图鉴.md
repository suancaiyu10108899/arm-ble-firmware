## 🎯 根因递进

### 扫描器 bug（v1.0-v1.4）
`Bluefruit.begin(peripheralCount, centralCount)` — **第一个参数是 Peripheral**。v1.3 全量日志证实 onScan 只触发一次后永久休眠。

### v1.5 突破：被动扫描 + loop 重启
`useActiveScan(false)` + loop 中每 10 秒 `Scanner.stop()` → `Scanner.start(0)` → 成功多次扫到 LOOKBON。

### GATT 发现不兼容（v1.6-v1.8）
Adafruit 库 GATT discover 与 LOOKBON 不兼容，底层协议交互失败。

### Python bleak 启示（v1.9-v2.4）
Python 代码 `ble_python.py` 跳过 discover，直接 `client.start_notify(7)`。

**v2.4 决定性修复**：将 CCCD 写入从 onConnect（SoftDevice 回调中断上下文）移到 loop（主循环），每次间隔 2 秒温和写入 → **数据通道打通**。

### 协议解析修复（v2.5-v2.6）
v2.4 数据 `raw=A5` 是单字节 0xA5，不是 ASCII "A5"。**LOOKBON 协议：高 nibble = 事件类型 (A=单击/B=长按/C=释放/D=方向)，低 nibble = 按键/方向编号**。

| 字节 | 含义 |
|------|------|
| 0xA1-0xA7 | 单击 @/A/B/C/D/R/L |
| 0xB1-0xB7 | 长按 |
| 0xC1-0xC7 | 释放 |
| 0xD0-0xD8 | 摇杆方向 (0=中位, 1-8=8方向) |
