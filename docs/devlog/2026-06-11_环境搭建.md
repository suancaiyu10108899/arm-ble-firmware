## 板子连接步骤（下次上手操作）

1. 找到一根**数据 MicroUSB 线**（插电脑后设备管理器出现新 COM 口 = 数据线 OK）
2. 线一头插电脑 USB，另一头插板子 MicroUSB 口
3. **不需要额外供电**——USB 口同时供电和编程
4. VS Code 终端运行 `python -m platformio run --project-dir=d:\Dev\arm-ble --target upload`
5. 板载红色 LED (pin 13) 开始 1Hz 闪烁 = ✅ 环境打通
